## 說明

這份文件說明 Claude 助教目前可以直接操作的 Google 工具。
這些工具透過 MCP（Model Context Protocol，模型連接協議）連接，
就像在 Claude 身上插了一個「Google 插頭」，可以直接讀取和操作你的 Google 帳號。

---

## 一、Gmail（電子郵件）

Claude 可以對你的 Gmail 做以下事情：

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「幫我找上週 BNI 相關的信」 | 搜尋符合條件的信件 |
| 「幫我讀這封信」 | 讀取並摘要信件內容 |
| 「幫我草擬一封回信給XXX」 | 建立草稿（Draft），你確認後再發送 |
| 「我的信箱有哪些標籤（分類）」 | 列出所有 Gmail 標籤 |
| 「幫我找有附件的信」 | 搜尋含附件的郵件 |

**注意：Claude 可以建立草稿，但不會直接發送信件，需要你最終確認。**

---

## 二、Google 日曆（Calendar）

Claude 可以對你的 Google 日曆做以下事情：

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「我這週有哪些行程？」 | 列出本週所有行程 |
| 「幫我安排明天下午三點的會議」 | 建立新的行程 |
| 「幫我找下週哪個時間有空」 | 分析空閒時間 |
| 「我有幾個日曆帳號？」 | 列出所有日曆 |
| 「幫我修改那個會議時間」 | 更新既有行程 |
| 「幫我接受/拒絕那個邀請」 | 回應會議邀請 |
| 「幫我找一個大家都有空的時間」 | 尋找共同空閒時段 |

---

## 三、Google 雲端硬碟（Drive）

Claude 可以對你的 Google Drive 做以下事情：

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「幫我找上個月的 BNI 報告」 | 搜尋雲端硬碟裡的檔案 |
| 「幫我讀這份 Google 文件」 | 讀取文件內容 |
| 「NotebookLM 裡加入這份雲端文件」 | 把 Drive 文件加進 NotebookLM |

---

## 工作範例（大樹教練常用情境）

### 情境一：BNI 會議前準備
「幫我找最近三封關於真誠分會的信，整理成摘要」
→ Claude 搜尋 Gmail → 讀取內容 → 整理摘要輸出

### 情境二：行程安排
「幫我看下週有沒有空，安排一個兩小時的 AI 學習時間」
→ Claude 讀取日曆 → 找空閒時段 → 建立行程

### 情境三：文件查找
「幫我在雲端硬碟找 4 月的燈號表」
→ Claude 搜尋 Drive → 回傳檔案名稱與連結
