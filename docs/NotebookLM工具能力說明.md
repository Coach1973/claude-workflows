# NotebookLM 工具能力說明
# 適用於：大樹教練（CoachWu）的 Claude Code 助教系統
# 更新日期：2026-04-06

---

## 說明

NotebookLM 是 Google 推出的 AI 筆記本工具。
Claude 助教透過 MCP 插頭直接連接你的 NotebookLM 帳號，
可以幫你建立筆記本、新增資料、產生簡報、Podcast 等內容，
不需要你手動開網頁操作。

```
你說話 → Claude 助教 → MCP 插頭 → NotebookLM → 產出內容
```

---

**切換帳號指令：**
- 切到公共帳號：`nlm login switch public`
- 切回個人帳號：`nlm login switch default`

---

## 一、筆記本（Notebook）管理

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「列出我所有的筆記本」 | 顯示全部筆記本清單 |
| 「幫我建一個叫 XXX 的筆記本」 | 新建筆記本 |
| 「幫我把 XXX 筆記本改名為 OOO」 | 重新命名 |
| 「把 XXX 筆記本刪掉」 | 刪除筆記本（需確認）|
| 「XXX 筆記本裡有什麼？」 | 列出筆記本的資料來源 |

---

## 二、資料來源（Source）管理

可以把各種資料丟進筆記本，讓 NotebookLM 學習：

| 來源類型 | 說明 | 你說的話範例 |
|----------|------|------------|
| 網址（URL）| 把網頁內容加進去 | 「把這個網址加進 XXX 筆記本」 |
| 文字（Text）| 直接貼上文字 | 「把這段文字加進去」 |
| Google 文件（Drive）| 從雲端硬碟加入 | 「把那份 Google 文件加進去」 |
| 本地檔案（File）| 上傳 PDF 或文字檔 | 「把這份 PDF 加進去」 |

---

## 三、產生內容（Studio）

這是 NotebookLM 最強大的功能，可以把筆記本內容轉成各種格式：

| 內容類型 | 說明 | 你說的話範例 |
|----------|------|------------|
| **簡報（Slides）** | 生成投影片，可匯出 .pptx | 「幫我用 XXX 筆記本做一份簡報」 |
| **Podcast 音頻** | 生成兩人對話式音頻 | 「幫我把 XXX 做成 Podcast」 |
| **影片概覽（Video）** | 生成影片摘要 | 「幫我做一個影片概覽」 |
| **資訊圖表（Infographic）** | 視覺化重點整理 | 「幫我做一張資訊圖表」 |

**注意：生成內容需要幾分鐘，Claude 會幫你等待並告知完成。**

---

## 四、查詢筆記本內容

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「問 XXX 筆記本：這份資料的重點是什麼？」 | 向筆記本提問，取得答案 |
| 「跨筆記本比較 A 和 B 的差異」 | 同時查詢多個筆記本 |
| 「幫我整理 XXX 筆記本的重點筆記」 | 產生摘要筆記 |

## 五、筆記（Note）管理

| 你說的話 | Claude 會做的事 |
|----------|----------------|
| 「在 XXX 筆記本新增一則筆記」 | 建立筆記 |
| 「更新那則筆記內容」 | 修改筆記 |
| 「刪除那則筆記」 | 刪除筆記（需確認）|

## 本地資料夾結構

NotebookLM 產出的內容會存放在：

```
（指定位置）/NotebookLM/
├── slides/          ← 簡報 (.pptx)
├── audio/           ← Podcast 音頻
├── video/           ← 影片概覽
├── infographics/    ← 資訊圖表
├── docs/            ← Google 文件匯出
├── mindmaps/        ← 心智圖
└── quizzes/         ← 測驗題
```
