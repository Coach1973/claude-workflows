## 正式名稱
- 對外口語：**頂級助教**
- 系統內部：**頂級特助**
- 命名日：2026-04-11，由大樹教練親自發想

## 命名核心洞見
「老闆不需學技術，只需出一張嘴」。市場上所有人教的是工具箱，這套系統賣的是一位能聽懂人話、全自動執行任務的頂級特助。

## 三套餐商業模式

| 套餐 | 名稱 | 目標客群 | 內容 |
|------|------|----------|------|
| A | 純享版 | 小白老闆 | Mac mini + OpenClaw 預裝 + 核心技能包，完全隱藏 Claude Code |
| B | 開發版 | 愛折騰玩家/學員 | Claude Code 建置與除錯教學 |
| AB | 終極版 | 高階訂製企業 | A+B 全開，老闆可呼叫 Claude 客製化新技能 |

## 護城河哲學
「任何人都可以複製今天的小龍蝦，但他們複製不了教練明天產生的大腦靈感。」
金句：*「你大可拿走我昨天的程式碼，因為我明天的自動化流程已經跑到你前面了。」*

**收費邏輯**：訂閱/俱樂部制，每週推送教練測試過最穩定的最新企業自動化技能包。

## 雙引擎分工
- **小龍蝦**：前台 24 小時執行者，老闆唯一對話窗口，能自己寫程式自己擴充技能
- **克勞德**：隱形急救工程師，只在小龍蝦當機或出 bug 時出場

## 執行紀律（2026-04-11 永久生效）
- 直接動手，不問同不同意
- 唯二例外：預估耗時 2~3 天以上；預估超過 5 萬 Token（10 萬需取得同意）
- 每隔幾小時回報進度百分比

## 市場定位
競品都在教「工具箱」。頂級特助的差異化：老闆不需學技術，只需出一張嘴——目前市場的絕對藍海。

**Why:** 這是整套系統的靈魂文件，包含命名由來、商業模式、守則，是偉大任務的核心支柱
**How to apply:** 所有決策、回應、系統設計都要對齊海餅乾守則精神，特別是「自動自發」與「全力以赴」
